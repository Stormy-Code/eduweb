Help

How to use

Type your message in the Encode tab. The Morse code appears below. Use Play to hear it, Download to save as MP3 or WAV, and Share to send to others.

Practice Mode

Go to the Practice tab to improve your skills. Choose from different drill types: single characters, words, call signs, or listen-and-type exercises. Your progress is tracked automatically.

Keyboard shortcuts

  • Ctrl+Enter or Cmd+Enter — Play / Stop
  • Esc — Stop playback
  • Enter — Submit answer (in Practice mode)

Decode

Paste Morse code (dots and dashes) in the Decode tab. Use · or . for dots, − or - for dashes. Use / or multiple spaces for word breaks.

Audio Profiles

  • Clean Tone — Pure sine wave
  • Radio Style — Simulated radio with slight noise
  • Telegraph — Sharp square wave beeps
  • CW Operator — Classic amateur radio sound

Reference

Click any character in the Reference tab to insert it into your message. Use the search box to quickly find specific characters.

Offline Mode

This app works offline! Install it to your device for the best experience.
